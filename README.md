# nafld-longitudinal-hdl-analysis
Longitudinal analysis of HDL cholesterol trajectories among individuals with Non-Alcoholic Fatty Liver Disease (NAFLD) using Linear Mixed-Effects Models and Generalized Estimating Equations in SAS.
Data are publicly available from Kaggle:

Non-Alcohol Fatty Liver Disease (NAFLD) Dataset

Source:
https://www.kaggle.com/datasets/utkarshx27/non-alcohol-fatty-liver-disease

Files used:
- nafld1.csv
- nafld2.csv

The raw datasets are not included in this repository.
